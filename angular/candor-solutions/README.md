# Candor Solutions – Internes Projektmanagement-Tool

## Über das Projekt

Candor Solutions GmbH ist eine kleine IT-Beratungsfirma. Dieses interne Tool
ermöglicht es Mitarbeitern, Kunden zu verwalten, laufende Projekte zu überblicken
und neue Beratungsanfragen einzureichen.

## Technologie

- Framework: Angular 21 (Standalone Components, Signals)
- Sprache: TypeScript
- Build-Tool: Angular CLI / Vite

## Schnellstart

```bash
git clone [URL]
cd candor-solutions
npm install
npm start
```

Der Entwicklungsserver läuft auf `http://localhost:4200`.
